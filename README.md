# Face Recognition Attendance System

## 📌 Project Overview
A Python-based Face Recognition Attendance System that automatically detects and recognizes faces using OpenCV and the face-recognition library, and records attendance.

## 🚀 Features
- Real-time face detection
- Face recognition using pre-trained models
- Automatic attendance marking
- Simple and clean Python implementation

## 🛠️ Technologies Used
- Python
- OpenCV
- face-recognition library
- NumPy

## 📂 Project Structure
- attendance.py → Main application file
- requirements.txt → Dependencies list

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the program:
   python attendance.py
